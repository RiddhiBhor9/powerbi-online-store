# powerbi-online-store
📊 E-Commerce Sales Analysis Dashboard (Power BI)
📌 Overview

This project presents an interactive Power BI dashboard designed to analyze and visualize e-commerce sales performance. It helps uncover key business insights such as sales trends, customer behavior, and product performance.

🎯 Objectives
Analyze overall sales performance
Identify top-performing products and categories
Track revenue trends over time
Understand customer purchasing behavior
Provide actionable insights for decision-making
📊 Dashboard Features
📈 Sales Overview – Total revenue, profit, and quantity sold
🛍️ Product Analysis – Best-selling products and categories
🌍 Regional Insights – Sales distribution by location
📅 Time Analysis – Monthly/Yearly trends
👥 Customer Insights – Purchase patterns and segmentation
🛠️ Tools & Technologies
Power BI – Data visualization
Excel / CSV – Data source
DAX – Data modeling and calculations
📂 Project Files
shree commerce.pbix – Main Power BI dashboard file
🚀 How to Use
Download the .pbix file
Open it in Power BI Desktop
Explore interactive dashboards and filters
